# hr-attrition-analysis
SQL-based analysis of employee attrition patterns.
**HR Attrition Analysis Using SQL**
This project analyzes employee attrition patterns using SQL to identify key factors influencing workforce turnover.

**Tools Used:**
MySQL
SQL (Joins, Aggregations, Subqueries)

**Key Insights:**
Higher attrition observed in R&D and Sales
Overtime employees show higher attrition
Lower tenure and lower salary are associated with higher turnover
